## Overview

The `Segment` and `Token` types represent transcribed audio with timing information. A `Segment` represents a complete phrase or sentence, while `Token` represents individual words within that segment.

## Segment

Represents a segment of transcribed audio with timing information. May include detailed token-level (word-by-word) information when available.

### Type definition

```typescript
type Segment = Token & {
    tokens?: Token[];
};
```

### Properties

<ResponseField name="text" type="string" required>
  The transcribed text for this segment.
</ResponseField>

<ResponseField name="start" type="number" required>
  Start time of the segment in seconds.
</ResponseField>

<ResponseField name="end" type="number" required>
  End time of the segment in seconds.
</ResponseField>

<ResponseField name="confidence" type="number">
  Confidence score for this transcription (between 0 and 1).
  
  Higher values indicate greater confidence in the transcription accuracy.
</ResponseField>

<ResponseField name="tokens" type="Token[]">
  Word-by-word breakdown of the transcription with individual timings.
  
  <Expandable title="Token structure">
    Each token in the array contains:
    - `text`: The individual word
    - `start`: Start time of the word in seconds
    - `end`: End time of the word in seconds
    - `confidence`: Optional confidence score for the word
  </Expandable>
</ResponseField>

## Token

Represents a token (word or phrase) in the transcription with timing information.

### Type definition

```typescript
type Token = TimeRange & {
    confidence?: number;
    text: string;
};
```

### Properties

<ResponseField name="text" type="string" required>
  The transcribed text (word or phrase).
</ResponseField>

<ResponseField name="start" type="number" required>
  Start time in seconds.
  
  Inherited from `TimeRange` type (from ffmpeg-simplified).
</ResponseField>

<ResponseField name="end" type="number" required>
  End time in seconds.
  
  Inherited from `TimeRange` type (from ffmpeg-simplified).
</ResponseField>

<ResponseField name="confidence" type="number">
  Confidence score for this transcription (between 0 and 1).
</ResponseField>

## Usage examples

<CodeGroup>

```typescript Basic usage
import { transcribe } from 'tafrigh';

const segments = await transcribe('audio.mp3');

// Access segment data
segments.forEach((segment) => {
  console.log(`[${segment.start}s - ${segment.end}s]: ${segment.text}`);
  console.log(`Confidence: ${segment.confidence}`);
});
```

```typescript Working with tokens
import { transcribe } from 'tafrigh';

const segments = await transcribe('audio.mp3');

// Access word-level timing
segments.forEach((segment) => {
  console.log(`Segment: ${segment.text}`);
  
  if (segment.tokens) {
    segment.tokens.forEach((token) => {
      console.log(`  - "${token.text}" at ${token.start}s`);
    });
  }
});
```

```typescript Filtering by confidence
import { transcribe, Segment } from 'tafrigh';

const segments = await transcribe('audio.mp3');

// Filter segments with high confidence
const highConfidence = segments.filter(
  (segment: Segment) => (segment.confidence ?? 0) > 0.8
);

console.log(`High confidence segments: ${highConfidence.length}`);
```

```typescript Generate subtitles
import { transcribe, Segment } from 'tafrigh';

const segments = await transcribe('audio.mp3');

// Convert to subtitle format
const subtitles = segments.map((segment: Segment, index: number) => {
  const startTime = formatTime(segment.start);
  const endTime = formatTime(segment.end);
  return `${index + 1}\n${startTime} --> ${endTime}\n${segment.text}\n`;
}).join('\n');

function formatTime(seconds: number): string {
  const hours = Math.floor(seconds / 3600);
  const minutes = Math.floor((seconds % 3600) / 60);
  const secs = Math.floor(seconds % 60);
  const ms = Math.floor((seconds % 1) * 1000);
  return `${pad(hours)}:${pad(minutes)}:${pad(secs)},${pad(ms, 3)}`;
}

function pad(num: number, size = 2): string {
  return num.toString().padStart(size, '0');
}
```
